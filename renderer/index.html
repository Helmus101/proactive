Frontmost OCR Capture
Capture text from the frontmost window every 30 seconds with Apple Vision OCR, keep the image ephemeral, and store only structured context.
Screen Recording Permission
macOS requires Screen Recording permission for frontmost-window OCR capture.
How OCR Works
Proactive uses a multi-stage OCR pipeline to turn the text in your frontmost window into searchable memory.
1. Frontmost Window Snapshot
Every few seconds while you are active, Proactive captures the frontmost window instead of blindly scanning the whole desktop. The image is treated as a temporary OCR input and is discarded after text extraction.
2. macOS Vision OCR
Text is extracted with Apple Vision on-device. The OCR pass finds visible text regions, applies language-aware recognition, and keeps confidence scores so weak reads can be filtered out.
3. Structure + Context
OCR text is combined with macOS Accessibility data, window title, app name, and timestamp so the result is more than a bag of words.
Layer What it provides
OCR (Vision) Literal text visible in custom UI, images, and surfaces without native text labels.
Accessibility API Structural roles such as buttons, fields, URL bars, and other interface semantics.
Context Wrapper App name, window title, timestamp, and frontmost-window metadata.
4. Memory Processing
Repeated states are deduplicated, sensitive patterns can be filtered before storage, and the resulting text is indexed so the memory graph can retrieve meaning instead of just exact keywords.
